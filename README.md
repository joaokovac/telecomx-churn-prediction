# Telecom X – Previsão de Churn de Clientes

Este projeto tem como objetivo analisar o comportamento dos clientes de uma empresa de telecomunicações e desenvolver um modelo capaz de prever a probabilidade de cancelamento de serviços (churn). A análise foi realizada utilizando técnicas de ciência de dados e machine learning, buscando identificar padrões nos dados que indiquem quais clientes apresentam maior risco de evasão.

O dataset utilizado neste projeto foi previamente tratado na primeira etapa do desafio Telecom X. A partir desses dados foram realizadas etapas de análise exploratória, com o objetivo de compreender melhor as características do conjunto de dados, observar a distribuição das variáveis e identificar possíveis relações entre elas. Durante essa etapa também foram gerados gráficos que ajudam a visualizar o comportamento dos clientes e a relação de algumas variáveis com o churn.

Após a análise inicial, os dados foram preparados para a criação do modelo preditivo. Isso incluiu a separação das variáveis utilizadas para treinamento do modelo, a conversão de variáveis categóricas e a divisão do conjunto de dados em dados de treino e teste. Também foi realizada a normalização das variáveis numéricas para melhorar o desempenho do algoritmo utilizado.

Para prever o churn foi utilizado um modelo de regressão logística. O modelo foi treinado utilizando parte dos dados e posteriormente avaliado com o conjunto de teste. A avaliação foi feita por meio de métricas como acurácia, relatório de classificação e matriz de confusão, permitindo observar o desempenho do modelo na identificação de clientes com maior probabilidade de cancelamento.

Este projeto demonstra como técnicas de análise de dados podem ajudar empresas a compreender melhor o comportamento de seus clientes e apoiar decisões estratégicas. Ao identificar padrões relacionados ao churn, a empresa pode criar ações de retenção mais eficientes e melhorar o relacionamento com seus clientes.
